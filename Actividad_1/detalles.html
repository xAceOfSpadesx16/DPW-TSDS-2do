Descripción del Proyecto

Este portfolio web fue desarrollado como trabajo práctico para el parcial del primer cuatrimestre de la materia "Diseño de Páginas Web". El proyecto reúne todas las actividades realizadas hasta la Actividad 3, organizadas en una estructura semántica coherente que simula un pequeño sitio web.

Estructura del Sitio Web

Páginas Incluidas:

  • index.html: Página principal con descripción general del portfolio
  • actividad1.html: Enlaces y formato de texto (Actividad 1)
  • actividad2.html: Listas, imágenes y contenido multimedia (Actividad 2)
  • actividad3.html: Tablas y formularios (Actividad 3)
  • detalles.html: Esta página con información técnica del proyecto

Implementación de Etiquetas Semánticas HTML5

Estructura Principal:

  • <header>: Encabezado general del sitio presente en todas las páginas
  • <nav>: Menú de navegación que conecta todas las páginas
  • <main>: Contenido principal de cada página
  • <footer>: Pie de página con información de copyright

Estructura de Contenido:

  • <section>: Secciones temáticas dentro del contenido
  • <article>: Contenido independiente y autocontenido
  • <aside>: Información complementaria
  • <figure>: Contenido multimedia con descripción

Contenido de las Actividades

Actividad 1 - Enlaces y Formato:

  • Estructura básica de HTML5
  • Etiquetas de formato: <em>, <strong>, <mark>, <del> <s>
  • Tres tipos de hipervínculos: interno, externo, entre páginas
  • Tres atributos diferentes: title, target="_blank", accesskey

Actividad 2 - Multimedia:

  • Listas ordenadas, desordenadas y de definición
  • Lista de términos con imágenes representativas
  • Dos archivos de audio con múltiples fuentes
  • Video con subtítulos en formato .vtt
  • Galería de imágenes adaptable con <picture>
  • Contenido embebido: YouTube, Google Maps, PDF

Actividad 3 - Tablas y Formularios:

  • Dos tablas con estructura semántica completa
  • Uso de <thead>, <tbody>, <tfoot>
  • Etiquetas <caption> para contexto
  • colspan y rowspan correctamente implementados
  • Celdas de cabecera con atributo scope
  • Formulario completo con múltiples tipos de inputs
  • Elementos de formulario avanzados y validación

Características Técnicas Implementadas

  • Accesibilidad: Atributos aria-current, labels asociados, estructura semántica
  • Responsive Design: Meta viewport, imágenes adaptables
  • SEO: Meta description, title descriptivo
  • Compatibilidad: Múltiples fuentes de audio/video, imágenes, etc
  • Navegación: Menú en todas las páginas, enlaces internos y externos
  • Validación: Formulario con atributos required, type, pattern, multiple, size, etc
